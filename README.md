# Arduino radio
## Dependencies:
Arduino:
- LiquidCrystal
- PU2CLR_SI470X
- Radio
- TimerOne

Other:
- minicom
- arduino-cli
## Compiling & uploading to Arduino
Edit the Makefile to your setup, then run
```bash
make compile upload
```
If you'd also want to see what the Arduino is doing, run
```bash
make monitor
```
